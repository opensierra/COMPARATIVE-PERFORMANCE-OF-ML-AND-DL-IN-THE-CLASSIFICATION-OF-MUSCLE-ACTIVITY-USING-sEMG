# COMPARATIVE-PERFORMANCE-OF-ML-AND-DL-IN-THE-CLASSIFICATION-OF-MUSCLE-ACTIVITY-USING-sEMG
Este estudio compara técnicas de aprendizaje automático y profundo en la clasificación de actividad muscular usando señales EMG del bíceps. Incluye metodologías, resultados y análisis para aplicaciones en rehabilitación y prótesis.
